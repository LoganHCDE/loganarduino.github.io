Code used


// An array to hold the pin numbers for the LEDs.
int ledPins[] = {13, 12, 11};

// The total number of LEDs in array.
int ledCount = 3;

// Global Delay
int animationDelay = 150;

// Setup function
void setup() {
  // Loop through each of the pins
  for (int i = 0; i < ledCount; i++) {
    // Set each LED pin as an output.
    pinMode(ledPins[i], OUTPUT);
  }
}

// The loop function runs over and over again forever.
void loop() {
  // --- Scan Forward ---
  // Loop from the first LED to the last.
  for (int i = 0; i < ledCount; i++) {
    digitalWrite(ledPins[i], HIGH); // Turn the current LED on.
    delay(animationDelay);          // Wait for a moment.
    digitalWrite(ledPins[i], LOW);  // Turn the current LED off.
  }

  // --- Scan Backward ---
  // Loop from the second-to-last LED back to the second one.
  for (int i = ledCount - 2; i > 0; i--) {
    digitalWrite(ledPins[i], HIGH); // Turn the current LED on.
    delay(animationDelay);          // Wait for a moment.
    digitalWrite(ledPins[i], LOW);  // Turn the current LED off.
  }
}

Answer to questions

  1. Draw a graph where the X axis is time and the Y axis is voltage across the LEDs.

    Draw a line for each of your 3 LEDs showing the voltage pulses as they turn on and off in sequence (each LED's line will be a series of square pulses staggered in time to match the scan forward and backward behavior).

  2. How many LEDs could you blink independently with your Arduino? How much current would that draw?

    There are 14 sections for connecting unique LEDs. That would draw about 280 milliampere (assuming ~20 mA per LED).

  3. How fast do you need to blink your LEDs until you no longer can tell that they are blinking?

    At least 30 to 60 times a second (30–60 Hz), which aligns with common perceptual frame rates where flicker becomes difficult to notice.
